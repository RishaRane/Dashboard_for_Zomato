Dataset Link : https://www.kaggle.com/datasets/himanshupoddar/zomato-bangalore-restaurants

NGORK site to generate Autorization Token : https://ngrok.com/


**PROEDURE:**

1. Sign in to `ngrok`
2. Generate an `Authorization Token` by selecting the appropriate operating system from the ngrok dashboard.
3. Insert the Authorization Token into the notebook to establish a `secure tunnel`.
4. Begin developing the Streamlit application and implement Python code for data visualization, using `Plotly` for interactive visualizations.
